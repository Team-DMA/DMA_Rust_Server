           [Инструкция по установке]

В данном случае, предполагается, что заказчик использует программу RustEdit. 
И так, файл у вас на компьютере. Далее...
1) Распакутйе архив в папку. 
2) Откройте расположение программы RustEdit.
3) Поместите папку с файлами префаба в папку "CustomPrefabs".
4) Запустить программу RustEdit.
5) Открыть карту, на которую вы хотите установить данный префаб.
6) Во вкладке Prefab List открыть вкладку Custom.
7) Во вкладке Custom открыть папку, которую мы перетаскивали в CustomPrefabs (п. 3).
8) Перетащить префаб в нужное вам на карте место.
9) Применить Модификации (!!!САМЫМ ПЕРВОМ ОБЯЗАТЕЛЬНО ПРИМЕНЯТЬ Height Mask, затем все остальное!!!).
10) Все готово! Осталось только сохранить карту, запустить на сервере и радоваться :)

-------------------------------------------------------------------------------------------------------

1P.S. Если RustEdit у вас уже запущен, то вы можете загрузить префаб 
не выходя из программы. После переноса папки с файлами префаба в CustomPrefabs, 
в программе, после нажатия вкладки Prefab List, справа вверху нажать на значок (в виде двух стрелочек).
После, префаб появится во вкладке Prefab List -> Custom, а дальше проделать пункты 7-12.

-------------------------------------------------------------------------------------------------------

2P.S. Для того, чтобы на вашем сервере работали кастомные IO цепочки и пазлы, кастомные ящики с лутом 
и кастомные магазины, вы должны установить расширение для оксида. (Имеется в наличие скаченного архива)


Переместите файл Oxide.Ext.RustEdit.dll по указанному пути:
Корневая папка сервера/RustDedicated_Data/Managed

           [Installation instruction]

In this case, it is assumed that the customer uses the RustEdit program. 
And so, the file is on your computer. Further...
1) Unpack the archive to a folder. 
2) Open the RustEdit program location.
3) Place the folder with the prefab files in the "CustomPrefabs" folder.
4) Run the RustEdit program.
5) Open the map on which you want to install this prefab.
6) in the Prefab List tab, open the Custom tab.
7) In the Custom tab, open the folder that we dragged to CustomPrefabs (item 3).
8) Drag the prefab to the place you want on the map.
9) Apply The Modifications (!!!THE first step is to APPLY the Height Mask, then everything else!!!).
10) Everything is ready! It remains only to save the map, run it on the server and be happy :)

-------------------------------------------------------------------------------------------------------

1P.S. If RustEdit is already running, you can download the prefab 
without leaving the program. After moving the folder with the prefab files to CustomPrefabs, 
in the program, after clicking the Prefab List tab, click on the icon at the top right (in the form of two arrows).
After that, the prefab will appear in the Prefab List -> Custom tab, and then go through steps 7-12.

-------------------------------------------------------------------------------------------------------

2P. S. in order for custom IO chains and puzzles to work on your server, custom boxes with loot 
and custom stores, you should install an extension for oxide. (Available in the downloaded archive)


Move the Oxide file.Ext.RustEdit.dll on the specified path:
The root folder of the server/RustDedicated_Data/Managed
